# Weather App

A Flutter project to learn the Bloc architecture. This app fetch the weather data from OpenWeather through REST API based on the user's current location.

## Packages Used
  * weather: ^3.1.1
  * intl: ^0.19.0
  * geolocator: ^11.0.0
  * flutter_bloc: ^8.1.5
  * equatable: ^2.0.5
  * flutter_dotenv: ^5.1.0
